About

Boom Boom Boom! is an audio visualisation App to liven up parties all over the world!

Throw up graphics, animations, text, photos and effects using your keyboard or by connecting a mobile device to use the custom controller. Or if you're feeling lazy just leave it to it's own devices and it plays in time with the music.

The App uses web-technologies including the Web Audio API to access and analyse audio from the microphone, Web-Sockets on a Node.js server to handle deviceâ€”desktop interaction, and CSS3 and Two.js to control animations.

Built as a Chrome Experiment by Jack Wild in Summer 2014.
